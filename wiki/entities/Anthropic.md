# Anthropic

> 一句话定义：Claude 系列大模型背后的 AI 安全公司，[[Andrej Karpathy]] 2026 年 5 月新东家，[[Memory Files]] / [[Dreams]] / [[Conway]] 三件套的推出方。

## 基本信息

- **类型**：公司 / AI 实验室
- **领域**：通用大模型、AI 安全
- **创始人**：Dario Amodei、Daniela Amodei 等（前 OpenAI 核心成员）
- **核心产品**：[[Claude Code]]、Claude 应用、Memory Files、Dreams、Conway（在研/灰度）
- **关联实体**：[[Andrej Karpathy]]（2026-05 加入）· [[OpenAI]]（人才源 + 竞争对手）

## 关键事实

- **2026-05-19** [[Andrej Karpathy]] 官宣加入 Anthropic 预训练团队，团队由前 OpenAI 核心 [[Nicholas Joseph]] 领导；Karpathy 将组建一个**用 [[Claude Code|Claude]] 加速 pretraining research** 的新团队（开启 [[递归自我改进]] 的工程化尝试）(来源: [[20260519-TechCrunch-Karpathy加入Anthropic预训练团队]] · 一手源)
  - ⚠️ 此前记录的 "2026-05-20" 系误记，已由一手源修正为 **05-19**（Tuesday）
- 同日宣布招募 **Chris Rohlf** 加入 frontier red team（前 Yahoo "The Paranoids" + Meta 安全 6 年）—— 与 Karpathy 招聘构成"进攻 + 防守对称"打法 (来源: [[20260519-TechCrunch-Karpathy加入Anthropic预训练团队]])
- 2026-05 推出 [[Memory Files]] / [[Dreams]] / [[Conway]] 三件套，标志 Claude 记忆能力最大规模架构翻新 (来源: [[20260525-知乎-Claude永久大脑真的来了]])
- 2026-03 底意外泄露 51.2 万行 [[Claude Code]] 源码，首次揭开 [[Conway]] 面纱
- Code with Claude 开发者大会（2026-05 旧金山）首次披露 Dreams
- 自家产品[[Claude Code]]在国外社区已有相当影响力（参见 [[forrestchang]] 的 [[CLAUDE.md]]，GitHub 15.5 万 star）
- **MCP 提出方**：Anthropic 不到一年前开源 [[MCP]]（Model Context Protocol），2026 年已成为 Agent ↔ 工具通信的事实标准，Google 跟进推出 [[A2A]]，IBM 推出 ACP，Linux 基金会成立智能体式 AI 基金会推动标准化 (来源: [[20260528-掘金-MCP协议落地三条铁律]])

## Dario Amodei 的核心判断

> ASI 不会是某一次突破的产物，而是一系列能力模块逐渐拼合的结果。

——记忆、推理、工具使用、代码执行、多模态感知都是拼图，记忆可能是最被低估的那一块。
