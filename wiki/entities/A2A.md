# A2A

> 一句话定义：Google 推出的 **Agent-to-Agent 通信协议**，定位与 [[MCP]] 形成竞争——MCP 解决 Agent 到工具，A2A 解决 Agent 到 Agent。

## 基本信息

- **类型**：开放协议
- **提出方**：Google
- **对应层**：Agent ↔ Agent（与 MCP 的 Agent ↔ Tool 互补/竞争）
- **关联实体**：[[MCP]] · [[Anthropic]]
- **关联概念**：[[Agent Gateway协议翻译层]]

## 当前状态

- 与 IBM ACP 一起被 Linux 基金会的"智能体式 AI 基金会"推动标准化(截至 2026-05) (来源: [[20260528-掘金-MCP协议落地三条铁律]])
- 实战中常和 MCP、REST、gRPC 等并存——这是 [[Agent Gateway协议翻译层]] 出现的根本原因

## 在本知识库中的角色

- **种子节点**：本库目前对 Google AI 生态笔墨极少，A2A 是第一个进入的 Google 协议产物
- 与 MCP 配对存在的"双协议世界"是 Agent 工程的现实生态
