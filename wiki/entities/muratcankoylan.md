# muratcankoylan

> 一句话定义：muratcankoylan 是 GitHub 项目 [[Agent Skills for Context Engineering]] 的作者/维护者，将 context engineering、harness engineering 与 multi-agent patterns 整理成可按需加载的 Agent Skills。

## 基本信息

- **类型**：开发者 / 开源项目作者
- **GitHub**：muratcankoylan
- **关联实体**：[[Agent Skills for Context Engineering]] · [[Hacker News]]
- **关联概念**：[[../concepts/上下文工程|上下文工程]] · [[../concepts/Skills 2.0模块化|Skills 2.0 模块化]] · [[../concepts/渐进式披露|渐进式披露]]

## 关键事实

- 其 HN 帖称过去一年构建 multi-agent systems 时反复遇到上下文窗口被工具输出填满、长对话中信息丢失、supervisor 累积 worker 状态变成瓶颈等问题 (来源: [[../summaries/20260607-HN-AgentSkillsForContextEngineering]])。
- 他将这些问题的解决方案整理为 Agent Skills，让 Agent 在相关任务中按需加载结构化指令 (来源: [[../summaries/20260607-HN-AgentSkillsForContextEngineering]])。
- 项目初始列出 7 个 skills，GitHub README 截至 2026-06-07 已扩展为 15 个 skills (来源: [[../summaries/20260607-HN-AgentSkillsForContextEngineering]])。

## 在本知识库中的角色

muratcankoylan 的贡献是把 [[../concepts/上下文工程|上下文工程]] 从理念/文章层推进到“可安装 Skill 库”层：不是只告诉 Agent 要管理上下文，而是给 Agent 一组可触发的上下文管理 SOP。
