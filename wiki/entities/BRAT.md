# BRAT

> 一句话定义：Obsidian 的 beta 插件加载器，让用户能装上未进入官方插件市场的第三方插件。

## 基本信息

- **类型**：Obsidian 插件
- **全称**：Beta Reviewers Auto-update Tool
- **侧边栏图标**：笑脸
- **安装方式**：Obsidian 官方插件市场直接搜索 "BRAT"
- **关联实体**：[[Obsidian]] · [[obsidian-agent-client]]

## 关键事实

- 通过仓库 URL 加载任意 GitHub 上的 Obsidian 插件 (来源: [[20260413-知乎-Qwen Code+Obsidian搭建Karpathy知识库]])
- 操作路径：BRAT → Plugins: Add a beta plugin for testing → 输入 Repository URL → 选 Latest version → Add plugin
- 本知识库工作流中，BRAT 是装 [[obsidian-agent-client]] 的前置依赖

## 在本知识库中的角色

仅在"方案 B"路径需要。当前"方案 A"无需安装。
